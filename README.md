# GermanSocorroAutomotivo

 
🚀 Socorro Automotivo 24h - Landing Page
Esta é uma landing page de alta conversão desenvolvida para um serviço de Socorro Automotivo na Vila Buarque e Grande São Paulo . O foco do projeto é a velocidade de carregamento e a facilidade de contato para usuários em situações de emergência.

📱 Funcionalidades
Design Responsivo: Adaptado para dispositivos móveis (foco principal para motoristas na rua).

CTA Rápido: Botões fixos de WhatsApp e Ligação para conversão imediata.

Seções Inclusas: Hero (Destaque), Sobre (Experiência), Serviços (Grid de Especialidades), Localização (Google Maps integrado) e Rodapé.

Dark Mode: Interface moderna com paleta de cores Grafite, Vermelho e Amarelo para passar autoridade.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica.

CSS3: Estilização avançada com Flexbox e CSS Grid.

API do Google Maps: Integração de mapa para localização física.
